# Laravel Project Rules

## General
- Create a complete web application using Laravel.
- Implement basic and advanced features.
- Use MVC, Eloquent ORM, migrations, and models.
- Secure the application with authentication and authorization.

## Authentication & User Management
- **Inscription & Connection**: Users must be able to register and log in.
- **Roles**: Two roles exist: `User` and `Administrator`.
- **Profile Management**: Users can update their profiles.
- **Admin Management**: Administrators can manage users (CRUD).

## Technical Requirements
- Use Laravel best practices.
- Ensure security best practices (CSRF, XSS protection, etc.).
- Maintain clean code structure.
